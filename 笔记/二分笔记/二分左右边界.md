### luoguP2249查找左右边界

> 1e6个有序数字,q个询问,每次询问$x$的最小下标,没有输出-1

* 二分左边界

  ```c++
  int lid() {
  	int ret = -1, lef = 1, rig = n, mid;
  	while(lef <= rig) {
  		mid = (lef + rig) >> 1;
  		if(a[mid] >= x) {
  			if(a[mid] == x) ret = mid;
  			rig = mid - 1;
  		} else 
  			lef = mid + 1;
  	}
  	return ret;
  }
  ```
